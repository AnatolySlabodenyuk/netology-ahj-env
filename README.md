# Netology AHJ — Environment Setup

Homework project for the "Working Environment" lesson.

## Stack

- Webpack 5
- Babel
- ESLint + Prettier
- Jest
- GitHub Actions CI/CD
- GitHub Pages

## Scripts

```bash
yarn start       # dev server
yarn build       # production build
yarn lint        # lint & fix
yarn test        # run tests
```
